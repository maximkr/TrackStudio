# Как копировать категорию

Вы можете "клонировать" одну или несколько выбранных категорий. При клонировании создаются новые категории, полностью идентичные клонируемым: для них сохраняются такие же настройки отображения, связи между категориями, настройки прав доступа, настройки триггеров и шаблоны.

Для того, чтобы клонировать категорию, перейдите через меню к списку категорий, отметьте нужную категорию и нажмите кнопку "**Клонировать**".
