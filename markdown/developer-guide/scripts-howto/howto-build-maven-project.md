# Как собрать проект из maven шаблона.

Для сборки проекта необходимо иметь установленный [maven](http://maven.apache.org/).

Заходим в папку etc/development/scripts и выполняем команду

```
mvn idea:idea
```

если вы используете IDE Intellij IDEA

```
mvn eclipse:eclipse
```

если вы используете Eclipse соответственно.

Теперь у вас есть готовый проект, который можно открыть в вашей любимой среде разработки.
