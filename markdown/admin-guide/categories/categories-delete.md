# Как удалить неиспользуемую категорию

Вы можете полностью удалить из системы неиспользуемые категории.

Для того, чтобы удалить категорию, перейдите через меню к списку категорий, отметьте нужную категорию и нажмите кнопку "**Удалить**".

![](../../images/categories_list.png)

Если категория используется хотя бы в одной из задач, вы получите сообщение об ошибке. В таком случае, вы можете [деактивировать эту категорию](categories-deactivate.md).
