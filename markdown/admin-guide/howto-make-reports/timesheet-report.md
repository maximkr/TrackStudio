# Табель

В табеле выводится количество часов, потраченных пользователем, на выполнение задач за указанный период. Фильтр при этом используется лишь для отсеивания ненужных задач. Вы можете построить этот отчет в форматах HTML, Excel и Word.

Для того, чтобы построить Табель

1. Перейдите в нужный проект
2. Затем через меню перейдите к отчетам.
3. Нажмите кнопку "**Табель**" в средней панели.

![](../../images/tabel.PNG)

1. Если вы хотите, чтобы вашим отчетом пользовались другие, отметьте опцию "**Общий**". Общие отчеты можно строить только на основе общих фильтров.
2. Выберите период за который нужно построить отчет
3. Укажите разбивку
4. Выберите пользователя или всех пользователей
5. Нажмите кнопку "**Сохранить**"

Сам отчет строится нажатием кнопки "Построить отчет". После нажатия кнопки вы получите либо страницу с отчетом, если был выбран формат HTML, либо файл с расширением, соответствующим выбранному формату.
