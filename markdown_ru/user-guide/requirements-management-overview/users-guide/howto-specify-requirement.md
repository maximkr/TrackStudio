# Как указать, для проверки какого требования предназначен набор тестовых данных

При создании набора тестовых данных укажите требование в соответствующем поле.

![](../../../images/clip0043.png)

Если же вам необходимо изменить этот параметр, перейдите к просмотру набора тестовых данных, затем нажмите Редактировать и укажите в соответствующем поле, какому требованию соответствует этот набор, после чего сохраните изменения.
