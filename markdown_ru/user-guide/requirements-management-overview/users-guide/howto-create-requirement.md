# Как создать требование

Перейдите к нужному вам списку требований. Нажмите кнопку **Создать требование** в верхней панели.

Укажите приоритет требования, его версию и техническое описание.

![](../../../images/clip0040.png)

Затем выберите нужные наборы тестов

![](../../../images/clip0041.png)
