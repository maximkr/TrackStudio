# Настройка прав на задачи с определенной категорией

В главе об управлении категориями задач мы уже описывали, [как настроить права пользователей для определенной категории](../categories/categories-permissions.md). При настройке новой или уже существующей роли вы можете выполнить те же действия, только уже для одной роли и нескольких категорий:

- Перейдите через меню к управлению пользователями
- Перейдите через меню к странице "**Роли**"
- Перейдите к нужной роли через список
- Выберите вкладку "**Разрешения по категориям**"
