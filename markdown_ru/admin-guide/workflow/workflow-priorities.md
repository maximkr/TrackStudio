# Как настроить набор приоритетов для процесса

Приоритет в задаче указывает на ее важность и определяет порядок обработки задач для пользователей. Набор приоритетов выводится пользователю в виде выпадающего списка при создании, редактировании задач, а также при выполнении операций. Каждому приоритету соответствует название и вес, выраженный простым числом. Этот вес учитывается при сортировке задач в списках. Чем больше число, тем выше приоритет.

Один приоритет из набора должен быть выбран в качестве приоритета по умолчанию. Для процесса также может не быть задано ни одного приоритета.

Для того, чтобы редактировать приоритеты в процессе, перейдите на страницу этого процесса через меню, а затем — на вкладку "**Приоритеты**".

## Как задать список приоритетов

Нажмите на кнопку "**Создать приоритет**" в средней панели. В открывшемся окне введите одно или несколько приоритетов. При вводе нескольких приоритетов задавайте их по одному на строку. Затем нажмите кнопку "**Сохранить**".

![](../../images/workflow_priorities_create.png)

## Как упорядочить список приоритетов

Вы можете упорядочить приоритеты для задач процесса, передвигая их в списке приоритетов с помощью кнопок. Также в этом списке вы можете поменять приоритет по-умолчанию. После внесения всех изменений нажмите кнопку "**Сохранить**" внизу формы.

![](../../images/workflow_priorities_edit.png)

## Как изменить название приоритета

Для того, чтобы изменить название приоритета, кликните на иконке ![](../../images/ico.edit_.gif) в строке нужного приоритета. В открывшейся форме введите новое название приоритета и, если нужно, измените его параметры.

![](../../images/workflow_priority_rename.png)

Нажмите кнопку "**Сохранить**" внизу формы.

## Как удалить приоритет

Для того, чтобы удалить один или несколько приоритетов, отметьте их флажком в списке и нажмите кнопку "**Удалить**" внизу формы.

Вы не сможете удалить из процесса приоритет, если он уже используется в существующих задачах.
