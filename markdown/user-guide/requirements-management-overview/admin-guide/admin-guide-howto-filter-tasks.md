# Как фильтровать задачи

Чтобы упорядочивать задачи в списке и отсеивать ненужные, вам и вашим пользователям понадобится набор предустановленных фильтров для задач. Создать фильтр для задач можно несколькими способами. Во-первых, можно перейти на страницу фильтров через меню **Управление задачами » Фильтры**. Обратите внимание, что фильтры привязаны к проекту и распространяются на подзадачи проекта вниз по иерархии. Т.е. фильтры, созданные для корневой задачи, будут видны во всех проектах TrackStudio.

![](../../../images/clip0030.png)

Нажмите кнопку **Создать фильтр** на панели. На появившейся странице заполните основные свойства фильтра. Выберите поля, которые будут выводиться в списке задач, обработанном этим фильтром. После нажатия на кнопку «Сохранить» появится окно со вкладками настройки всех параметров фильтра.

![](../../../images/clip0031.png)

Можно пойти другим путем: перейти к списку задач в нужном вам проекте. Выбрать любой фильтр на панели списка задач. Там же нажать **Настройки фильтрации**. С помощью выпадающего списка выбрать нужные вам параметры по одному, заполняя поля ввода и сохраняя выбор кнопкой **Установить**

![](../../../images/clip0032.png)

Затем нажмите кнопку **Применить** и посмотрите, те ли результаты вы хотели получить. Если настройка фильтра закончена, нажмите кнопку **Сохранить как фильтр**. Ваши настройки будут сохранены в **новом** фильтре, вам останется лишь поменять ему название.

Фильтры на странице со списком задач можно группировать. Группа задается в поле "Название фильтра" с помощью прямого слеша (/). Например "Мои задачи/Созданные мной", "Мои задачи/Нужно делать" и т.п.
