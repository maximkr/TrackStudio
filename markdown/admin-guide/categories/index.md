# Как определить и настроить категории задач

## Что такое категории?

Категории представляют собой типы задач, такие как группа проектов, проект, версия, компонент, сообщение об ошибке и др. Категории позволяют привязать к задачам процессы, организовать иерархию задач (проекты могут содержать сообщения об ошибках), настроить параметры отображения задачи и права пользователей.

## Создание новой категории

Для того, чтобы создать новую категорию, перейдите к управлению задачами. Затем перейдите на тот уровень иерархии задач, на котором вы хотите определить категорию. Задачи с этой категорией можно будет создавать только ниже по иерархии.

![](../../images/category_create.png)

- Перейдите в меню "**Управление задачами > Категории**".
- Нажмите кнопку "**Создать категорию**" в средней панели.
- Выберите название категории.
- Выберите нужный процесс.
- Заполните остальные поля.
- Нажмите кнопку "**Сохранить**".
