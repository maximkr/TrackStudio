# Как оповещать об изменениях по e-mail незарегистрированных пользователей

В TrackStudio есть возможность оповещения незарегистрированных в системе пользователей через СС.

Для этого необходимо настроить [оповещение для пользователей](howto-notify-users.md). Далее отредактировать выбранный в оповещении шаблон (шаблоны находятся в папке TrackStudio/etc/plugins/e-mail/) и добавить в него строку

```
<#assign CC="${task.getUdfValueByName(\"CC\")}"/>
```

При изменении шаблона перезапуск TrackStudio не требуется.

В процессе необходимо создать дополнительное поля строкового типа с названием "CC".

![](../images/CC_settings.PNG)

Для отправки оповещений достаточно указать e-mail адреса пользователей в дополнительном поле "СС". При этом необходимым условием является срабатывание оповещения для пользователя, который уже есть в системе.

![](../images/CC_emails.PNG)
