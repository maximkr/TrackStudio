# Как разделить пользователей на группы

Перейдите в меню **Управление пользователями » Роли**. В появившемся окне нажмите **Создать роль**, введите ее название и сохраните.

![](../../../images/clip0026.png)

Перед вами откроется страница с несколькими вкладками.

![](../../../images/clip0027.png)

Перемещаясь по вкладкам, настройте свойства и права доступа группы к операциям в системе.
