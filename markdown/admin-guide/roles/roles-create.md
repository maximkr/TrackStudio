# Как создать новую роль

Для регистрации новой роли пользователей в системе:

- Перейдите через меню к управлению пользователями
- Перейдите через меню к странице "**Роли**"
- Нажмите кнопку "**Создать роль**" в панели списка ролей
- В открывшемся меню введите название роли. Укажите опцию, если нужно поместить кнопку создания пользователя с этой ролью на панель
- Нажмите кнопку "**Сохранить**"

![](../../images/roles_create.png)

При создании роли для нее выставляются права, наследуемые от основной роли создателя. То есть если для вашей роли запрещен, например, просмотр ссылок SCM, то он также будет запрещен и для всей ролей, которые вы создадите. Однако вы можете затем разрешить для созданной роли эти действия.

## Как удалить роль из системы

Неиспользуемую роль можно удалить из системы. Для этого:

- Перейдите через меню к управлению пользователями
- Перейдите через меню к странице "**Роли**"
- Отметьте одну или несколько ролей, которые нужно удалить
- Нажмите кнопку "**Удалить**"

## Как копировать роль

Вы можете скопировать любую из созданных вами или вашими подчиненными ролей. При клонировании полностью копируются свойства роли, разрешения для нее в категориях и процессах, разрешения на выполнение действий и доступ к полям. Но **не копируются права доступа к проектам**. Для того, чтобы клонировать роль:

- Перейдите через меню к управлению пользователями
- Перейдите через меню к странице "**Роли**"
- Отметьте одну или несколько ролей, которые нужно клонировать
- Нажмите кнопку "**Клонировать**"
