# Как создать новую категорию

Перейдите в пункт меню **Управление задачами » Категории** и на открывшейся странице нажмите кнопку **Создать категорию** в панели Списка категорий, либо в панели быстрого доступа.

![](../../../images/clip0029.png)

Далее система переведет вас в окно обзора свойств категории. В первую очередь настройте **Связи между категориями**, укажите, в каких категориях могут создаваться задачи вашей категории. Перемещаясь по закладкам, установите **Права на категорию** (кто из пользователей имеет право работать с задачами данной категории); **Шаблон**, если вы хотите, чтобы создаваемые задачи имели определенный вид описания. Если вы хотите, чтобы при создании и редактировании задачи выполнялись какие-либо скрипты, укажите их на закладке **Триггеры**.
