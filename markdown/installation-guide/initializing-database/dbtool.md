# Использование консольной утилиты для экспорта, импорта и инициализации данных

По многочисленным просьбам в TrackStudio Enterprise 4.0.10 мы добавили консольную утилиту для экспорта и импорта базы данных **dbtool**. Теперь, даже не имея на сервере графической оболочки, вы можете получить анонимный файл экспорта с вашими данными для отправки в нашу службу поддержки.

Пользоваться этой утилитой очень просто:

```
Внимание! Утилита использует данные о соединении с базой, прописанные в файлах настроек TrackStudio. Перед использованием утилиты остановите сервер TrackStudio, если он запущен.
```

1. Перейдите в папку, в которую установлена TrackStudio
2. В консоли или в командной строке наберите

```
./dbtool export -a --file=export-a-20100129.xml
```

Вы получите анонимный файл экспорта.

Для того, чтобы получить экспортный файл с реальными данными:

```
./dbtool export --file=export-a-20100129.xml
```

Для того, чтобы импортировать ваши данные в новую базу данных:

```
./dbtool import --file=export-a-20100129.xml
```

Для того, чтобы инициализировать базу данных по-умолчанию на русском языке:

```
./dbtool import --file=./etc/resources/sql/LAST/ru_def.xml
```

Для того, чтобы инициализировать базу данных по-умолчанию на английском языке:

```
./dbtool import --file=./etc/resources/sql/LAST/en_def.xml
```
