# Настройка X Window

TrackStudio не содержат графические библиотеки для создания цветов, шрифтов и другой графической информации. Java использует системные библиотеки для получения такой информации, таким образом, для отображения графической информации (отчетов) вам потребуется графическая подсистема X Window.

- Установите X11 Server и задайте переменную окружения DISPLAY
- Передайте параметр **-Djava.awt.headless=true** виртуальной машине Java, когда запускаете ее. В этом случае запуск Xvfb не требуется, но само наличие пакетов X11 обязательно.

Для разных серверов приложений этот параметр задается по-разному. Например для Tomcat 5.x, вы можете прописать эти опции в **catalina.bat** или **catalina.sh** в **CATALINA_OPTS**.
