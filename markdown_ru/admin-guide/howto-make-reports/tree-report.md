# Отчет о структуре задач

С помощью отчета о структуре задач вы можете экспортировать данные в MS Project, либо в другую систему, поддерживающую формат XML, либо с помощью своего XSL-файла преобразовать информацию о структуре задач в вашем экземпляре TrackStudio в нужную вам форму.
