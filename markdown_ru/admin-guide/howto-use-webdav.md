# Как редактировать приложенные файлы через WebDav.

**В TrackStudio есть возможность редактировать приложенные файлы через WebDav.**

Данная возможность позволяет отредактировать приложенный в задачу файл и сохранить изменения сразу на сервере.

Для этого необходимо скопировать ссылку WebDav link

![](../images/copyLink.PNG)

Далее в редакторе Microsoft Word в меню открытия файла нужно вставить эту ссылку и открыть файл

![](../images/open.PNG)

Теперь все изменения, внесенные в документ, можно сохранять прямо на сервере.

Для редактирования файлов можно использовать Microsoft Office, XMLSpy и другие программы, поддерживающие работу с WebDav.
