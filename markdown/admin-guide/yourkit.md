# Как оценить быстродействие и использование памяти с помощью YourKit

YourKit — коммерческая утилита для оценки и оптимизации быстродействия программ, написанных на Java. Если вам кажется, что TrackStudio работает необычайно медленно, с помощью этой утилиты вы сможете собрать для нас необходимую информацию, а мы — исправить ситуацию и улучшить работу TrackStudio на вашей конфигурации.

Для того, чтобы оценить быстродействие TrackStudio, вам не понадобится покупать YourKit. На сайте YourKit можно скачать [пробную версию](http://www.yourkit.com/download/index.jsp) для вашей операционной системы. Просто запустите YourKit с сделайте следующее:

1. Подключитесь к процессу startJetty или jettyService.
2. Запустите Start CPU Profiling (Sampling).
3. Повторите проблемную ситуацию.
4. Снимите snapshot и пришлите его нам.
