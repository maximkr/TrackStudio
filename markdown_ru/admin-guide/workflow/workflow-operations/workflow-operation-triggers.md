# Как настроить триггеры для операций

Триггеры для операций — очень мощный инструмент для реализации ваших бизнес-процессов.

К каждой операции вы можете привязать свой набор триггеров, которые будут запускаться при выполнении операции. Триггеры могут быть трех типов: одни выполняются до основных действий (непосредственно перед выполнением операции и изменением состояния задачи), вторые — вместо основного действия и третьи — после выполнения действия. На одно действие, таким образом, вы можете назначить не более трех триггеров. Подробно о создании триггеров рассказано в соответствующем [разделе документации](../../../developer-guide/scripts-howto/index.md).

Для того, чтобы назначить на операцию триггер, перейдите на страницу этой операции из списка операций, а затем — на вкладку "**Триггеры**".

![](../../../images/workflow_operation_triggers.png)

Если в системе найдены триггеры определенного типа, они будут выведены для выбора. Если триггеров такого типа нет, поле выбора не появится.

После выбора нужных триггеров нажмите кнопку "**Сохранить**".
