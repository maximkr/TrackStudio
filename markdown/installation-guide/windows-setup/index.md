# Установка под Windows

Мы настоятельно рекомендуем скачивать и устанавливать TrackStudio Enterprise Standalone, используйте WAR версию только в исключительных случаях, т.к. она значительно сложнее в установке и использовании и не имеет особых преимуществ.

Вариант Standalone поставляется с утилитой для конфигурирования и запуска TrackStudio Server Manager. В комплект также входит сервер приложений jetty, почтовый сервер и СУБД Hypersonic SQL. Данный вариант работает фактически "из коробки", то есть вы можете после установки сразу же запустить TrackStudio без настройки, чтобы ознакомиться с системой.

Также Server Manager понадобится вам для экспорта и импорта данных из базы.

Вариант WAR предназначен для запуска под имеющимся у вас сервером приложений (обычно это Tomcat). Он содержит нескомпилированные JSP, которые Tomcat собирает сам. Этот вариант настраивается путем редактирования файлов properties. Вы также можете перенести предварительно настроенные в Server Manager файлы properties с другой машины.
