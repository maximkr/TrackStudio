# Как настроить переходы из одного состояния в другое

Для того, чтобы выполнение операции приводило к переводу задачи из одного состояния в другое, нужно для этой операции задать переходы.

Чтобы задать переходы, откройте страницу операции, нажав на ссылку в списке операций и выберите вкладку "**Переходы**".

![](../../../images/workflow_operation_transitions.png)

На открывшейся странице вы увидите два столбца с состояниями. В левом столбце — начальное состояние перехода, в правом — конечное. Одна операция может переводить задачу из разных состояний в разные, из разных в одно, но не из одного состояния — в разные.

Выберите одно или несколько состояний в левом столбце и только одно — в правом. Нажмите кнопку "**Создать**".

Созданные переходы появятся на этой же странице в списке ниже, а использованное начальное состояние исчезнет из левого столбца.

![](../../../images/workflow_operation_transitions_edit.png)

Чтобы удалить переход, выберите его в списке и нажмите кнопку "**Удалить**". Вы можете удалять переходы из операций в любое время, даже если эти операции уже выполнялись в задачах.
