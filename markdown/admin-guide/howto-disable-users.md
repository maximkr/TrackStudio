# Как отключить ненужных пользователей

В бесплатной версии TrackStudio есть ограничение на 5 активных пользователей в системе (в **Starter Kit** - 10 пользователей). Если у вас в системе создано больше пользователей, чем позволяет лицензия, вы можете отключить ненужных.

Для этого:

1. Войдите в систему администратором (или любым пользователем, который может редактировать учетные записи подчиненных)
2. Перейдите к учетной записи нужного пользователя
3. Откройте ее для редактирования
4. Снимите отметку "**Активный**"
5. Сохраните параметры пользователя

![](../images/disable_user.png)

Неактивные пользователи не могут войти в систему и их нельзя сделать ответственными за задачу, но все их данные остаются в системе. Тем же способом их можно будет позже снова активировать.

Эта возможность пригодится так же, если вы вдруг кого-то уволили. Учетную запись корневого пользователя деактивировать нельзя.

```
Мы настоятельно не рекомендуем удалять учетные записи пользователей. Используйте вместо удаления дезактивацию.
```
