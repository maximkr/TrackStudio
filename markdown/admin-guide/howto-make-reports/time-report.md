# Отчет о затраченном времени

В отчете о затраченном времени вы можете увидеть активность пользователей по задачам, выраженную в часах. Этот отчет будет вам полезен, если вы используете почасовую систему оплаты труда.

Чтобы построить отчет по активности за какой-то определенный период, используйте фильтрацию по дате выполнения операций.

Если вы хотите лишь посмотреть затраты по задачам, без детализации по пользователям, отметьте опцию **скрыть ответственного** в настройках отчета.

Также для этого типа отчетов можно построить круговую диаграмму, если отметить соответствующую опцию.
