# Как определить и настроить процессы

## Что такое процессы?

Процессы представляют собой жизненный цикл задачи.

В процессах описывается набор состояний, в которых может находиться задача, правила перехода из одного состояния в другое, набор приоритетов, дополнительных полей; также указывается кто может переводить задачу из одного состояния в другое.

## Создание нового процесса

Для того, чтобы создать новый процесс, перейдите к управлению задачами. Затем перейдите на тот уровень иерархии задач, на котором вы хотите определить процесс. Задачи с этой процессом можно будет создавать только ниже по иерархии. Можно создавать все процессы на уровне корневой задачи, а привязывать к конкретным узлам иерархии через категории.

![](../../images/workflow_create.png)

- Перейдите в меню "**Управление задачами → Процессы**".
- Нажмите кнопку "**Создать процесс**" в средней панели.
- Выберите название процесса.
- Нажмите кнопку "**Сохранить**".
