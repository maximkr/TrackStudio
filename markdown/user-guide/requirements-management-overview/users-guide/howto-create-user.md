# Как создать нового пользователя

Эти действия могут выполнять администраторы и менеджеры системы, причем существует иерархия сотрудников – добавлять и удалять разрешено, только подчиненных пользователей. Чтобы добавить нового пользователя, зайдите в раздел меню **Управление пользователями » Список пользователей**. На панели быстрого доступа нажмите кнопку для добавления пользователя с нужной ролью (также обратите внимание на меню **Другие действия** на панели).

Заполните открывшуюся форму. Вы можете сразу задать пароль для этого пользователя. Сам пользователь после входа сможет его сменить. Если вы укажете **проект по-умолчанию**, пользователь получит доступ к этому проекту.
