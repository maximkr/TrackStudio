# Как задать образец для оформления задачи

Иногда требуется, чтобы при создании новой задачи ее описание было выполнено по определенному образцу, содержало определенные пункты и было оформлено в соответствии с требованиями. Используйте для этого шаблоны категорий. Шаблон — это обычный HTML-документ, созданный во встроенном редакторе точно так же, как задается обычное описание задачи.

Для того, чтобы задать шаблон для определенной категории, перейдите на страницу этой категории, а затем — на вкладку "**Шаблон новых задач**". После того, как оформите шаблон в редакторе, нажмите кнопку "**Сохранить**".
