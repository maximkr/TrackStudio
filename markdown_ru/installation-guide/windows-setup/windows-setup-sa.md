# Установка TrackStudio Standalone под Windows

Для того, чтобы установить TrackStudio Standalone:

1. Запустите TrackStudio Enterprise Server Manager (**sman.exe**).
2. Выберите "Русский" в меню File->Language, если отображаются надписи на английском языке.
3. В меню нажмите кнопку "**Создать БД**"
4. В появившемся окне выберите вариант базы данных для инициализации.

![](../../images/database_create_database_sman_ru.png)

1. Нажмите кнопку "**ОК**"
2. После того, как база данных будет создана, нажмите кнопку **Старт**, чтобы запустить сервер TrackStudio.
3. Когда сервер запустится, нажмите кнопку "**Открыть в браузере**".

**Используйте следующие учетные записи:**

| **Пользователь** | **Группа** | **Логин** | **Пароль** |
| --- | --- | --- | --- |
| Администратор | Администратор | root | root |
| Сергей Менеджеров | Менеджеры | manager | 123 |
| Иван Аналитиков | Аналитики | analyst | 123 |
| Дмитрий Писателев | Технические писатели | writer | 123 |
| Степан Разработчиков | Разработчики | developer | 123 |
| Максим Тестеров | Тестеры | tester | 123 |

## Примечание

Если вы используете СУБД HSQLDB, файлы базы данных будут иметь вид *tsdb.lck, tsdb.log, tsdb.properties* and *tsdb.script.* Не удаляйте их.
