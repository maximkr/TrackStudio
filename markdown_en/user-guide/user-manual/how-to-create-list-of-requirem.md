# How to Create List of Requirements/ Bugs (Errors)/ Test Documentation/ Completions

Lists serve as containers (folders) for similar entities within the scope of a project. Lists are created in the same way as any other task in TrackStudio i.e. through buttons in the toolbar or dropdown menu.
