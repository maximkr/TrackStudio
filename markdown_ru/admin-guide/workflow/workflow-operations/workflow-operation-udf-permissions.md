# Как настроить разрешения для дополнительных полей в операциях

Если для процесса заданы дополнительные поля, вы можете настроить правила, по которым в определенной операции пользователи будут видеть эти дополнительные поля и иметь возможность их редактировать.

Для того, чтобы задать права для дополнительных полей в операциях, перейдите к нужной операции через список и выберите вкладку "**Разрешения для дополнительных полей**".

Настройки каждого из правил представлены в виде двух столбцов с полями выбора. Одно поле управляет отображением поля при выполнении операции, другое — возможностью его редактировать. Эти права распространяются **на всех пользователей**, независимо от роли, которые могут **выполнить** операцию.

![](../../../images/workflow_operation_udf_permission.png)

При создании операции автоматически выставляются разрешительные права для всех дополнительных полей, уже определенных в процессе. При создании дополнительного поля автоматически выставляются разрешительные права для всех операций.

Настройки правил не должны противоречить друг другу. Например, нельзя разрешить изменение поля, но не разрешить его просмотр.
