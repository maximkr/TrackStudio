Тренд

Отчет типа Тренд позволяет оценить количество новых, открытых, закрытых задач в каждый отчетный период. В отчете подсчитывается количество операций различных типов за период времени. Это может быть полезно, например, для определения готовности продукта к релизу. Данный отчет можно построить в пяти разных масштабах времени: годы, месяцы, дни, недели и часы. Отчет строится в форматах XML или CSV.

 

Вы можете построить этот отчет на основе любого фильтра. Для того, чтобы построить тренд:

 

1.Перейдите в нужный проект

2.Затем через меню перейдите к отчетам.

3.Нажмите кнопку "Тренд" в средней панели.

4.В открывшейся форме введите название отчета, выберите фильтр и масштаб времени (например, неделю)

5.Если вы хотите, чтобы вашим отчетом пользовались другие, отметьте опцию "Общий". Общие отчеты можно строить только на основе общих фильтров.

6.Нажмите кнопку "Сохранить"

 

 

После создания отчета откроется страница с его свойствами. На ней можно указать формат вывода отчета (CSV или XML), кодировку символов, разделитель для CSV.

Также можно временно изменить параметры фильтрации задач. Это может быть полезно, если нужно строить отчеты за разные интервалы времени.

 

 

Непосредственно сам отчет строится нажатием кнопки "Построить отчет". После нажатия кнопки вы получите файл с расширением csv. Этот файл можно открыть в табличном процессоре (Excel или Calc).

 

 

 

Далее постройте диаграмму:

В Calc через меню Вставка → Диаграмма выбирайте Гистограмму с накоплением

 

 

Выберите диапазон так, чтобы он охватывал все данные и подписи. Первый столбец и первая строка диапазона будут подписями, данные располагаются в столбцах.

 

После этого нажмите кнопку "Готово".
