# Распределение

Отчет типа **Распределение** показывает распределение заданной характеристики задач по одному или двум признакам. Например, количество задач, созданных определенным пользователем в зависимости от состояния задачи. Или распределение бюджетов по задачам разного типа.

Отчет типа **Распределение** может быть построен в форматах HTML, PDF, Excel и Word и содержать круговую диаграмму.

Допускается группировка данных по полям: Автор, Задача, Категория, Состояние, Ответственный, Приоритет, Резолюция, Роль автора, Роль ответственного, а также по дополнительным пользовательским полям. Отчет типа **Распределение** оперирует численными данными. Вы можете построить отчет о: распределении задач по группам, распределению бюджета, количеству операций, потраченному времени и количеству подзадач (если речь, например, идет о проектах). К этим данным можно применять функции: максимум, минимум, среднее и сумма.
