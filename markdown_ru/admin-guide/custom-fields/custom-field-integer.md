# Дополнительное поле типа Целое

## Свойства

Целое число, вводимое пользователем.

![](../../images/custom-field-integer-edit.png)

## Ввод

Целое число должно быть в пределах от –2147483648 до 2147483647. Значение этого поля соответствует типу Integer в Java. На форме редактирования представлено строкой ввода <INPUT type="text">. Вводимые пользователем в поле значения контролируются с помощью javascript. Значения, вводимые триггерами и иными средствами на уровне API также контролируются.

## Вывод

Значение этого поля выводится как строка при отображении в браузере. Если значение поля не задано, ни значение, ни заголовок поля не будут выводиться в просмотре задачи (пользователя). Значение вычисляемого поля задано всегда, но оно может оказаться пустым.

## Фильтрация и поиск задач

```
Внимание! Дополнительные поля, привязанные к задачам (пользователям) не отображаются для фильтрации в фильтрах, созданных на уровнях выше, где эти поля определены. Например, с помощью фильтра "Все задачи" нельзя выводить и фильтровать задачи по дополнительному полю, созданному на втором уровне иерархии.
Для того, чтобы искать задачи по дополнительному полю, вам нужно создать фильтр либо в той же задаче, где определено поле, либо в ее подзадачах. Удобнее всего скопировать для этого готовый фильтр.
```

Фильтрация задач по дополнительному полю типа **Целое** осуществляется по условиям:

- равно
- не равно
- больше или равно
- меньше или равно
- Входит в список

Условие "входит в список" задается набором чисел и числовых интервалов. Например, "**4, 8, 15, 16, 23-42**".
