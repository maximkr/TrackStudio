# Как создать проект или версию

Перейдите на корневую задачу (пункт "Проекты") в дереве задач. Нажмите кнопку **Создать проект** в верхней панели. В открывшейся форме заполните нужные вам поля (поле **Название** является обязательным). Можете указать планируемую длительность проекта как в виде бюджета времени, так и в виде сроков готовности.

После заполнения формы нажмите кнопку **Сохранить**

![](../../../images/clip0038.png)

Версия проекта создается аналогичным образом.
