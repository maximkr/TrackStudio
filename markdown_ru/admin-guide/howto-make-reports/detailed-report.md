# Детальный

В **детальном** отчете для задач выводятся значения всех видимых полей. Фильтр при этом используется лишь для отсеивания ненужных задач. Отчет представляет собой таблицу с параметрами задач. Вы можете построить этот отчет в форматах HTML, PDF, Excel и Word

Вы можете построить этот отчет на основе любого фильтра. Для создания отчета:

1. Перейдите в нужный проект
2. Затем через меню перейдите к отчетам.
3. Нажмите кнопку "**Детальный**" в средней панели.
4. Если вы хотите, чтобы вашим отчетом пользовались другие, отметьте опцию "**Общий**". Общие отчеты можно строить только на основе общих фильтров.
5. Нажмите кнопку "**Сохранить**"

![](../../images/report_detailed_create.png)

После создания отчета откроется страница с его свойствами. На ней можно указать формат вывода отчета (HTML, PDF, Excel и Word).

Также можно временно изменить параметры фильтрации задач.

![](../../images/report_detailed_properties.png)

Сам отчет строится нажатием кнопки "**Построить отчет**". После нажатия кнопки вы получите либо страницу с отчетом, если был выбран формат HTML, либо файл с расширением, соответствующим выбранному формату.

![](../../images/report_detailed_view.png)
