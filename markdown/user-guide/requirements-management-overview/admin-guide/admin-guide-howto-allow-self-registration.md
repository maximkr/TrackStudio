# Как разрешить пользователям регистрироваться самостоятельно

Если вы хотите, чтобы в вашем проекте участвовали или следили за ним "внешние" пользователи, вам понадобится сделать для них правило саморегистрации, по которому они смогут регистрироваться и заходить в проект.

Перейдите к Вашему проекту. Зайдите в пункт меню **Управление задачами » Правила регистрации**. На открывшейся странице нажмите кнопку **Создать правило регистрации**. Введите название правила и выберите роль, с которой будут регистрироваться новые пользователи.
