# Как запретить просмотр или редактирование полей пользователей

В TrackStudio вы можете управлять видимостью и доступностью для редактирования полей пользователей для каждой роли отдельно. Для того, чтобы настроить права на поля пользователей:

- Перейдите через меню к управлению пользователями
- Перейдите через меню к странице "**Роли**"
- Перейдите к нужной роли через список
- Выберите вкладку "**Доступ к полям пользователей**"

![](../../images/roles_user_field.png)

На этой вкладке представлена таблица с полями пользователей, а также списки дополнительных полей, привязанных к пользователям.

Для некоторых стандартных полей пользователей вы можете управлять и видимостью и редактированием (*Компания, Телефон*). Для некоторых только редактированием (*Роль, Почта, Локализация, Часовой пояс, Активный, Действителен до, Лицензировано пользователей, Проект по умолчанию, Шаблон e-mail по умолчанию*).

Вы можете управлять видимостью и редактированием дополнительных полей, привязанных к пользователям. Настройки каждого из правил представлены в виде таблицы с опциями "Разрешен просмотр" и "Разрешено редактирование".

Настройки правил не должны противоречить друг другу. Например, нельзя разрешить редактирование поля, но не разрешить его просмотр.
