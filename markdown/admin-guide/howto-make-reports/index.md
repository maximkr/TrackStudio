# Как построить отчеты в TrackStudio

В TrackStudio отчеты предназначены для представления данных. Все отчеты используют фильтры для выборки нужных задач. Всего существует 7 типов отчетов: **Список**, **Детальный**, **Распределение**, **Отчет о затраченном времени**, **Структура проекта**, **Тренд** и **Диаграмма Ганта**.

Чтобы создать отчет любого выберите в меню пункт **Отчеты**. На странице отчетов нажмите в панели кнопку, соответствующую типу отчета. Например, **Создать отчет в виде списка**. Задайте название, выберите нужный вам фильтр и параметры формата вывода. Затем сохраните параметры отчета. После этого на странице просмотра выберите нужный формат и нажмите кнопку "Построить отчет". Ваш отчет откроется на отдельной странице.

Также на странице просмотра вы можете менять настройки фильтра для отчета без сохранения их обратно в фильтр - точно так же, как в списке задач.

На странице просмотра приводится ссылка, по которой можно переходить к отчету напрямую, минуя этап настроек. В этой ссылке слово PASSWORD нужно заменить на пароль.

Для построения отчета от анонимного пользователя ни логин, ни пароль указывать не нужно.

Для некоторых отчетов (например, Список) вы можете задавать ширину выводимых колонок в пикселях. Для этого найдите и отредактируйте файл **reports.properties** в папке **webapps/TrackStudio/WEB-INF/classes**.
