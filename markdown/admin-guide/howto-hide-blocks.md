# Как скрыть ненужные блоки со страницы отображения задачи

В TrackStudio версии 4.0.15 мы к каждому блоку на странице свойств задачи, в котором выводится информация, присвоили индивидуальный ID. Теперь вы можете скрывать эти блоки, если информация из них вам не нужна. Для этого отредактируйте файл **style.css**, расположенный в папке **webapps/TrackStudio**.

В конце файла добавьте код, скрывающий блок. Например, такой:

```
#divLastMessage {
display: none;
}
```

Список блоков:

| **divLastMessage** | Блок, в котором выводится последняя операция |
| --- | --- |
| **chatDiv** | Блок с чатом |
| **divScm** | Блок вывода информации из систем управления исходным кодом, если они подключены |
| **divAttachment** | Блок с приложенными файлами |
| **logoHeader** | Логотип |
