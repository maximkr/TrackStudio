# Как задать набор резолюций

Резолюции можно использовать для указания стандартизованных причин выполнения операции. Например, задача может быть закрыта с резолюциями "исправлена", "устарела", "не воспроизводится" и подобным. Установить резолюцию для операции можно при ее выполнении, выбрав нужную из выпадающего списка. При этом резолюция будет зафиксирована и для задачи, к которой относится операция.

Чтобы задать резолюции, откройте страницу операции, нажав на ссылку в списке операций и выберите вкладку "**Резолюции**". Затем нажмите кнопку "**Создать резолюцию**" в панели.

![](../../../images/workflow_operation_resolution_create.png)

В открывшемся окне введите название резолюции. Можно также установить флажок "По умолчанию" — тогда данная резолюция будет автоматически выбираться при выполнении операции. Нажмите кнопку "**Сохранить**"

Для того, чтобы изменить название резолюции, кликните на иконке ![](../../../images/ico.edit_.gif) в списке резолюций. В открывшейся форме введите новое название резолюции.

Чтобы удалить резолюцию, выберите ее в списке и нажмите кнопку "**Удалить**" внизу формы.

![](../../../images/workflow_operation_resolutions_list.png)
