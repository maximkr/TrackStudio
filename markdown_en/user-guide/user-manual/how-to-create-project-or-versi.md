# How to Create Project or Version

Go to the root task (item ‘Projects’) in the Tasks Tree. Press the button **Create Project**** **in the upper toolbar. In the subsequent form, fill the required fields (field **Name** is mandatory). You can indicate the scheduled duration of the project in the form of time budget as well as in the form of deadlines.

After filling the form, press the button **Save**.
