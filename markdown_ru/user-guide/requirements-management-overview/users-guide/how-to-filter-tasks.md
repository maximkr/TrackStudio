# Как отфильтровать список задач

Списки задач в TrackStudio выводятся с учетом параметров, заданных в **Фильтрах**. Для того, чтобы сменить фильтр, просто нажмите его кнопку на панели над списком, либо выберите из выпадающего меню.

![](../../../images/clip0035.png)

Для более тонкой настройки фильтров можно использовать форму ниже

![](../../../images/clip0036.png)

Также в **Фильтрах** поддерживаются [регулярные выражения](http://docs.oracle.com/javase/7/docs/api/java/util/regex/Pattern.html).

Например для того чтобы отфильтровать задачи по непустому полю "Описание", в настройках фильтрации задач выбранного фильтра используйте регулярное выражение ".+" как на скриншоте.

![](../../../images/filter.PNG)

Регулярное выражение (.*)(bugs)(.*) проверяет наличие слова bugs в выбранном поле, а выражение [^(bugs)]+ будет проверять поле на отсутствие слова bugs.
