# Диаграмма Ганта

Вы можете использовать [диаграмму Ганта](http://ru.wikipedia.org/wiki/%D0%94%D0%B8%D0%B0%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0_%D0%93%D0%B0%D0%BD%D1%82%D0%B0) в планировании ваших проектов в качестве иллюстрации графика работ как на этапе планирования, так и на этапе реализации плана.

Каждый отрезок диаграммы соответствует одной задаче. Отрезок начинается в точке, соответствующей дате и времени создания задачи и заканчивается в точке ее последнего обновления (последней активности по задаче). Если для задачи указаны сроки выполнения, они также будут отмечены

Данный вид отчета может быть построен в форматах HTML, PDF, Excel и MS Word.
