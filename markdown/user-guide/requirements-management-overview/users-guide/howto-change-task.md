# Как изменить состояние задачи, ответственного, сроки выполнения и т.п.

Зайдите в соответствующую задачу. На ее панели выберите кнопку с нужным вам действием. Например **Закрыть задачу** или **Добавить комментарий**

![](../../../images/clip0033.png)

В открывшемся окне заполните описание операции и измените значения нужных вам полей.

![](../../../images/clip0034.png)
