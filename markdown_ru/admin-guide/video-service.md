# Как использовать TrackStudio в качестве видеосервиса

В версии 5.5.0 появилась возможность использовать TrackStudio как видеосервис.

Теперь вы можете добавлять свои видео-файлы в задачи и комментарии. Просмотр видео осуществляется в полнофункциональном плеере с функциями перемотки, выбора скорости воспроизведения и регулировкой громкости.

## Как добавить свое видео в задачу.

- Для начала необходимо загрузить видео-файл в одну из задач. Как вариант, видео-файлы можно хранить в отдельной задаче типа "Контейнер файлов".
- Для вставки видео в комментарий либо описание задачи используется макрос

```
Video{task_number:attach_id}
```

task_number - это номер задачи, к которой приложен видео-файл.

Для вставки макроса также можно воспользоваться пунктом Add Video из меню Tools в редакторе.

![](../images/tools.PNG)

AttachmentId, необходимый для вставки, можно узнать из ссылки на файл.

![](../images/CopyLink.PNG)

```
http://localhost:8888/TrackStudio/download/task/124/**8a80828f58f2efc20158f31b2763004e**
```

Video{124:8a80828f58f2efc20158f31b2763004e}

В задачу будет вставлен видео-файл с AttachmentId 8a80828f58f2efc20158f31b2763004e из задачи #124.
