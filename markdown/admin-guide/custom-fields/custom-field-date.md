# Дополнительное поле типа Дата

## Свойства

Дата и время, вводимые пользователем.

![](../../images/custom-field-date-edit.png)

## Ввод

Значение поля представляет собой дату и время и соответствует типу Calendar в Java. На форме редактирования представлена строкой ввода <INPUT type="text">, снабженную всплывающим календарем для выбора даты. Формат даты и времени соответствует региональным настройкам пользователя в TrackStudio. Вводимые пользователем в поле значения контролируются с помощью javascript. Значения, вводимые триггерами и иными средствами на уровне API также контролируются.

## Вывод

Значение этого поля выводится как форматированная строка при отображении в браузере. Если значение поля не задано, ни значение, ни заголовок поля не будут выводиться в просмотре задачи (пользователя). Значение вычисляемого поля задано всегда, но оно может оказаться пустым.

## Фильтрация и поиск задач

```
Внимание! Дополнительные поля, привязанные к задачам (пользователям) не отображаются для фильтрации в фильтрах, созданных на уровнях выше, где эти поля определены. Например, с помощью фильтра "Все задачи" нельзя выводить и фильтровать задачи по дополнительному полю, созданному на втором уровне иерархии.
Для того, чтобы искать задачи по дополнительному полю, вам нужно создать фильтр либо в той же задаче, где определено поле, либо в ее подзадачах. Удобнее всего скопировать для этого готовый фильтр.
```

Фильтрация задач по дополнительному полю типа **Дата** осуществляется по условиям:

- позже заданной даты
- ранее заданной даты
- указанное количество минут (часов, дней или месяцев) назад или раньше
- указанное количество минут (часов, дней или месяцев) назад или позже
- указанное количество минут (часов, дней или месяцев) вперед или раньше
- указанное количество минут (часов, дней или месяцев) вперед или позже
