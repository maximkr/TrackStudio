# Как временно отключить неиспользуемые категории

Неиспользуемые категории можно отключить. Отключение категории означает, что нельзя будет создавать новые задачи с этой категорией, но уже созданные задачи никуда не исчезнут. Таким же образом можно контролировать создание задач определенных категорий (например, проектов) только администратором, либо только в определенное время (например, версий).

Для того, чтобы отключить категорию, перейдите на страницу этой категории через меню и нажмите кнопку "**Деактивировать**" на панели свойств категории. Чтобы включить категорию обратно, нажмите кнопку "**Активировать**" там же.
