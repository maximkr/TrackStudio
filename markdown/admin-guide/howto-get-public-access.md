# Как организовать доступ к TrackStudio без регистрации

В TrackStudio 4.0.8 появилась возможность организации анонимного (публичного) доступа для пользователей без регистрации непосредственно через интерфейс TrackStudio. Ранее такая возможность была только в интерфейсных шаблонах, которые нужно было подключать к проектам через предварительно созданные учетные записи. Теперь же вы можете дать незарегистрированным пользователям ограниченный вами самими доступ к TrackStudio через web-интерфейс, либо через SOAP API (REST API).

Анонимный доступ осуществляется через специального пользователя anonymous с ролью "Анонимные пользователи". Управлять правами анонимного пользователя вы можете точно так же, как и правами обычных пользователей: через настройку прав для его роли и непосредственно для пользователя.

Чтобы анонимные пользователи имели доступ к какому-то проекту, нужно [разрешить его](roles/howto-define-roles-and-groups/index.md) точно так же, как и обычным пользователям. При этом вы можете свободно использовать механизм переназначения ролей. На разные проекты публичный доступ может быть организован по разному.

Чтобы отключить публичный доступ к вашему экземпляру, просто деактивируйте пользователя anonymous.
