# Как настроить связи между категориями

Для того, чтобы использовать созданную категорию, вы должны связать ее с уже существующими категориями. Для этого перейдите через меню "Управление задачами -> Категории" на страницу созданной вами категории и выберите вкладку "**Связи между категориями**".

Вы увидите два списка. Перенесите в правый список те категории, в которых вы ходите создавать задачи с вашей. Например, обычно задачи можно создавать внутри проектов и версий, а проекты внутри задач — нельзя. Папки обычно могут быть вложены друг в друга, а ошибки — нет.

После изменения списков нажмите кнопку "**Сохранить**".
