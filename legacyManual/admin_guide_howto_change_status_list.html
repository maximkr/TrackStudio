Как изменить список состояний у задачи

Список состояний задачи определяется ее Процессом. Перейдите в пункт меню Управление задачами » Процессы. Из списка выберите нужный процесс и нажмите закладку Состояния.

 

 

Чтобы создать новое состояние задачи, нажмите кнопку Создать состояние на панели списка состояний.

Вы можете создать сразу несколько состояний, разместив их по одному на строчке в поле ввода

 

 

Просто создания состояния мало. Для того, чтобы его можно было использовать, нужно встроить его в существующие переходы между состояниями. Т.к. переходы между состояниями в TrackStudio выполняются с помощью операций, нам лучше создать новый тип операции (это не обязательно, мы можем встроить переход в уже имеющиеся операции, но в данном случае создать новый тип будет интереснее и полезнее)

Переходим на закладку Типы операций и нажимаем на панели кнопку Добавить операцию

 

После создания типа операции вы перейдете на страницу с его свойствами. Там на вкладках можно настроить Резолюции, Переходы,Триггеры, Разрешения для операций и Разрешения для дополнительных полей. Нас в первую очередь интересуют Переходы.

 

Выберите на форме, из какого состояния в какое должна переходить задача в этой операции. Например из Новое в Зарезервировано.

 

 

После создания перехода Вы можете также настроить разрешения. По-умолчанию этот переход будет доступен для выполнения всем пользователям, котором доступна задача. Вы можете ограничить как выполнение операции, так и ее просмотр (Например, чтобы скрыть внутреннюю переписку с заказчиком или, наоборот, между разработчиками)
